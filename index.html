1. Basic HTML document

Page section Semantics Description
N/A <!DOCTYPE html> To declare the doc type at the start of the file
N/A <html>All html content</html> All HTML content to be placed within this tag
Head <title>Page Title Here</title> Show Page Title in browser window (not shown on page)
<link href="style.css" type="text/css" rel="stylesheet"> To link CSS file to HTML file
Body <main>Content here</main> Page content to be input here
Footer <footer>Footer here</footer> Footer content (eg. contact info, copyright info...) to be input here

2. Common HTML Tags

Name Semantics Description
Image <img src="url.jpg" alt="text describing image"> Self-closing tag. Alt attribute is important for accessibility & SEO
Video <video src="video.mp4" controls>Video not supported</video>
  • Controls attribute will display video controls in the media player
  • The content inside the opening & closing tags is shown as a fallback in browsers that don't support the element
Audio <audio src="video.mp4" controls>Video not supported</audio>
  • Controls attribute will display video controls in the media player
  • The content inside the opening & closing tags is shown as a fallback in browsers that don't support the element
Anchor <a href="random-url.com" target="_blank">
  • Create hyperlinks
  • Target="_blank" attribute will make the hyperlink opened in a separate browser tab
Figure & Caption <figure>
<img src="image.jpg">
<figcaption>Description of the media</figcaption>
</figure>
  • Used to encapsulate media (image, diagram or code snippet)
  • Figcaption element is used to describe the media

3. Table

Name Semantics Description
Table <table></table> The wrapper element for all HTML tables
Table Head <thead></thead> The set of rows defining the column headers in a table
Table Body <tbody></tbody> The set of rows containing actual table data
Table Row <tr></tr> The table row container
Table Data <td></td> The table data container,making up individual cell
Table Foot <tfoot></tfoot> The set of rows defining the footer in a table
Colspan <td colspan="2"></td> Defines how many columns a td element should span
Colspan <td rowspan="2"></td> Defines how many rows a td element should span

4. Basic & Common CSS

Name Semantics Description
Style by element p { font-weight = 12; } Element tag followed by curly brackets
Style by class .class-name { font-weight = 12; } Dot, followed by class name, followed by curly brackets
Style by id #id-name { font-weight = 12; } Hash, followed by id (should be unique), followed by curly brackets
